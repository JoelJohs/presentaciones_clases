### Cómo identificar la arquitectura

```bash
# Linux
lscpu | grep Architecture
# x86_64 → 64 bits | i686 → 32 bits

uname -m
# Mismo resultado, más conciso

# Windows
msinfo32  →  buscar "Tipo de sistema"
```

### Problemas comunes en equipos legacy
- **Ventoy no funciona** en BIOS muy antiguas porque su bootloader espera UEFI o BIOS moderno
- **Windows 11 requiere UEFI + Secure Boot** oficialmente, pero Tiny11 omite ese requisito
- **Discos >2 TB en BIOS:** No se pueden usar como disco de arranque con MBR

## 3. Creación de USB Booteable para Legacy

En esas máquinas Ventoy no funcionó. `dd` es el método universal.

### Paso a paso desde Linux

```bash
# 1. Identificar el USB (SIN confundirte)
lsblk

# 2. Si está montado, desmontarlo
sudo umount /dev/sdX1

# 3. Escribir la ISO directo al dispositivo (sin número)
sudo dd if=tu-iso.iso of=/dev/sdX bs=4M status=progress

# 4. Esperar que termine y sincronizar
sync
```

> ⚠️ **CUIDADO:** `of=/dev/sdX` es el dispositivo completo (sin número de partición). Si pones `sda` en vez de `sdb`, borras tu disco principal.

### ¿Por qué `dd` sí funciona?
`dd` copia la ISO bit por bit, boot sector MBR incluido. La ISO ya trae código de arranque legacy. Ventoy instala su propio bootloader y las BIOS viejas no lo reconocen.

## 6. Repaso Rápido del Módulo 1

### Conceptos clave que vimos
1. **Hardware vs Software** — Lo físico vs lo digital. Si lo tocas es hardware.
2. **Sistema Operativo** — Traductor entre hardware y usuario. Gestión de procesos, memoria, archivos, dispositivos.
3. **Componentes internos** — CPU (chef), RAM (mesa), SSD/HDD (mochila), PSU, Motherboard
4. **Periféricos** — Entrada (teclado, mouse), Salida (monitor, bocinas), Mixtos (pantalla táctil)
5. **Archivos y extensiones** — `.pdf`, `.mp3`, `.mp4`, `.exe`. Peligro: extensiones ocultas
6. **Terminal CMD** — `dir`, `cd`, `mkdir`, `color`, `tree`, `ver`
7. **Mantenimiento Preventivo** — Limpieza, pasta térmica, desfragmentación (solo HDD)
8. **Mantenimiento Correctivo** — Diagnóstico, reemplazo, restauración
9. **USB Booteable** — ISO, Rufus/Ventoy/`dd`, formateo e instalación del SO
10. **Troubleshooting** — No enciende, no da video, BSOD, PC lenta
